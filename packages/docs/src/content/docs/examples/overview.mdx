Hoppity ships with four example projects in the `examples/` directory. Each demonstrates a different messaging pattern and middleware combination. All examples include Docker Compose files for RabbitMQ and work out of the box with default settings.

## Prerequisites

All examples require:

- Node.js 22+
- pnpm 9+
- Docker (for RabbitMQ)

```bash
# From the repo root
docker compose up -d    # Start RabbitMQ
pnpm install            # Install all workspace dependencies
```

## Pattern Comparison

| Pattern          | Example         | When to Use                                                      |
| ---------------- | --------------- | ---------------------------------------------------------------- |
| Pub/Sub          | basic-pubsub    | Simple event broadcasting, decoupled consumers                   |
| Delayed Messages | delayed-publish | Scheduled notifications, retry with backoff, deferred processing |
| RPC              | rpc             | Synchronous request/response between services, queries           |
| Contract-Driven  | bookstore       | Multiple services with shared domain contracts, typed messaging  |
